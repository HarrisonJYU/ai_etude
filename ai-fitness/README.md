# Local development

for yt dl to work locally you need to have `ffmpeg` installed

on mac

```shell
brew install ffmpeg
```

on ubuntu/debian

```shell
sudo apt-get install ffmpeg
```

on Windows

```shell
choco install ffmpeg
```
